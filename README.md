中京大学スクールバスの**非公式**アプリです。

## Requirements
- CocoaPods
- Carthage

## Installation
**Firebase**を使用しているため  `GoogleService-Info.plist` が必須です。  
用意した上でプロジェクトがあるディレクトリで以下のコマンドを実行してください。

```
make bootstrap
```
